# Teams lunch break setter
Script em Python para automatizar a definição de status e mensagem de ausência no intervalo para o almoço. Define o status para "Volto logo", com duração de uma hora, coloca uma mensagem de ausência com o cálculo do horário de retorno.

## Dependências:

````
python -m pip install mouse
python -m pip install keyboard
python -m pip install datetime
````

É necessário inserir as coordenadas iniciais para que o mouse mova até o ícone do usuário e o script comece a executar. Uma sugestão para descobrir as coordenadas corretas é tirar um print da tela em que o Teams está aberto, abrir o print no Paint e posicionar o mouse sobre o ícone do usuário e anotar as coordenadas:
